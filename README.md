# WooSale Customizer AJK

A powerful WordPress plugin that enhances WooCommerce sale badges with 10 modern, customizable styles and full translation support.

## Features

- **10 Modern Styles**: Choose from minimalist, gradient, pulsating, shadow, rounded, diagonal, icon-based, transparent, neon, and premium classic designs
- **Custom Text**: Replace the default "Sale" label with any text you want
- **Live Preview**: See your changes in real-time before saving
- **Full Translation Support**: Built-in Polish and English translations, with WordPress translation system integration
- **WooCommerce Compatible**: Fully tested with WooCommerce 3.0+ and declared compatible up to version 9.0
- **Easy Integration**: Seamlessly integrates into WooCommerce admin menu

## Installation

1. Upload the `woo-sale-customizer-ajk` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to **WooCommerce → Sale Customizer AJK** to configure

## Configuration

1. Go to **WooCommerce → Sale Customizer AJK** in your WordPress admin
2. Enter your custom sale label text
3. Select one of the 10 available styles
4. Use the live preview to see how it looks
5. Click **Save** to apply changes

## Requirements

- WordPress 5.5 or higher
- WooCommerce 3.0 or higher
- PHP 7.0 or higher

## Translation

The plugin includes Polish and English translations by default. Additional translations can be added through WordPress translation system using the `.pot` file located in the `languages` folder.
