# Changelog

All notable changes to WooSale Customizer AJK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-27

### Added
- Initial release
- 10 modern sale badge styles (minimalist, gradient, pulsating, shadow, rounded, diagonal, icon-based, transparent, neon, premium classic)
- Custom sale label text functionality
- Live preview in admin settings page
- Full translation support (Polish and English)
- WooCommerce menu integration
- WooCommerce compatibility declaration
- WordPress translation system integration (.pot, .po, .mo files)

### Features
- Style selection dropdown with 10 options
- Real-time preview of selected style and text
- Override default WooCommerce sale badge
- Admin settings page with intuitive interface
- Translation-ready codebase
